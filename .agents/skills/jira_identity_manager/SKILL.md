# jira_identity_manager

## Objetivo
Administrar identidades dentro de Jira, asegurando que se respete la regla de "Permission Minimalism".

## Tareas principales
- Crear usuarios y definir grupos (ej. `safe-art-leads`).
- Mapear estos grupos organizacionales a **Roles de Proyecto**.
- Nunca asignar permisos directamente a usuarios individuales o grupos globales.

## Ejemplo de uso
"Configura el acceso para un equipo de 10 personas bajo el rol de 'Business Owners'."
